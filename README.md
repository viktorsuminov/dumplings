В работе использовал пакетный менеджер uv если у вас он ещё не установлен, то выполните команду: 

# Windows
powershell -c "ir | iex" (или через pip install uv)
# macOS/Linux
curl -LsSf https://astral.sh | sh

Для автоматического создания виртуального окружения и установки зависимостей достаточно команды:
uv sync

Далее запустить приложение можно командой:
uv run uvicorn main:app --reload

Чтобы запустить контейнер с базой выполните команду:
docker compose up -d
